les etape que j'ai pris pour construire mon site E-commerce a l'aide du CMS "wordPress" : 
/////////////////////////////////////////////////////////////////////////////////

* j'ai utiliser le theme astra ce theme instal 2 plugin inportant pour crée un site e-commerce woocommerce et elementor 
			1) woocommerce : 
				WooCommerce est une extension open source pour WordPress permettant
 				de créer une boutique en ligne. Il est conçu pour les petites
 				et grandes entreprises en ligne utilisant WordPress. Lancé 
 				le 27 septembre 2011, le plugin est rapidement devenu célèbre
				pour sa simplicité d'installation et de personnalisation.

			1-A) jai utiliser ce plugin pour onstruire mon site web ecommerce, jai choisie woocommerce 
				parce que il offre un enorme suite de controle sur to 
				les espét du boutique (gestion de produits, gestion de coupon ...).
				j'ai aussi trover la manubilation des paramettre dans ce plugin vraiment simple.

			2) elementor :
				Elementor est l'éditeur de glisser-déposer WordPress ultime et
 				gratuit, qui vous permet de créer de magnifiques sites
 				Web WordPress de la manière la plus simple et la plus rapide possible.

			2-A) jai utiliser ce plugin pour modifier le theme que jai utiliser pour construire 
				mon site e-commerce et le rendre plus adapter a mes propre utilisations et mes gouts
				jai aussi trouvé ce plugin vraiment simple a gerer.

			3) yoast : 
 				Yoast facilite le réglage des publications pour l’optimisation
 				SEO grâce à l’analyse des contenus publiés. L’outil a retenu
 				des critères mesurables pour optimiser le référencement d’un
 				contenu sur Google. Il s’intéresse à la lisibilité du contenu
 				d’une part, et à la place du mot clé dans le contenu d’autre part.

			3-A) ce plugin demande de completé un formulaire concernant vetre site et
				il c'aucupe de l'otimisation de votre site web si il a un probleme 
				il va vous prevenir et demande de la resoudre 
			
			4) GTranslate : 
				que plugin aide à traduire là un site web une langue vers une autre choisie
 				ce plugiciel et gratuit et aussi offre plusieurs langues à traduire contrairement 
				à d'autre plugins qui offrent une seule langue avec option de paiement si on veut traduire 
				à plusieurs langues

			5) CookieYes : Le Règlement général sur la protection des données (RGPD),
 				entré en vigueur le 25 mai 2018, renforce les droits
 				des citoyens de l'Union européenne concernant leurs
 				données personnelles. Toutes les entreprises et associations
 				des 28 États membres, ainsi que celles issues des pays hors
 				UE mais collectant et traitant des données de résidents européens
 				(comme Google, Facebook ou Amazon par exemple), y sont soumises.
